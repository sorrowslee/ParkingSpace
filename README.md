# ParkingSpace - 停車場視覺化與抽選推薦系統

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Technology](https://img.shields.io/badge/Technology-Fabric.js-orange.svg)

這是一個專為大樓住戶設計的地下停車場視覺化工具，旨在幫助住戶在年度停車位抽選時，能快速掌握空位資訊並做出最佳決策。

## 🚀 快速啟動 (Windows)

本專案提供了一個自動化腳本，讓您可以一鍵啟動服務：

1. 確保您的電腦已安裝 [Node.js](https://nodejs.org/)。
2. 在專案資料夾中，連點兩下執行 **`StartParkingTool.bat`**。
3. 程式會自動啟動本地伺服器並開啟瀏覽器訪問 `http://localhost:5500`。

---

## 🛠️ 核心功能

### 1. 繪圖模式 (Editor Mode)
- **🅿️ 停車位**：點擊建立車位，選中後可透過上方圓點自由旋轉角度。
- **⬜ 柱子/牆**：按住滑鼠左鍵可「刷」出整面牆。柱子會自動對齊 40px 的網格。
- **🛣️ 道路**：自由曲線繪製路徑。
- **🛗 電梯與🏠家電梯**：標記公共設施與住家入口位置。
- **🧹 橡皮擦**：按住並塗抹可連續刪除物件。
- **💾 JSON 管理**：支援「另存新檔」與「載入存檔」，讓您的地圖數據永不遺失。

### 2. 抽選模式 (Lottery Mode)
- **實時更新**：輸入已被抽走的車位號碼（如 `237`）點擊標記，該位置會立即變紅並排除出清單。
- **✨ 推薦系統**：系統會自動依評價 (A > B > C > D) 排序，為您列出當前剩下的最優位置。
- **精準定位**：點擊推薦列表中的號碼，畫布會自動**縮放並移至畫面中心**，伴隨閃爍提醒。

## 🖱️ 快捷操作
- **右鍵拖曳**：在任何工具狀態下，按住右鍵即可拖曳地圖看板。
- **Alt + 拖曳**：同樣具備拖曳地圖的功能。
- **滑鼠滾輪**：縮放地圖大小。
- **雙擊車位**：修改車位號碼與評價（具備重複號碼檢查機制）。

## 🌐 遠端與行動存取

### 1. 同區域網路 (Wi-Fi)
當您執行 `StartParkingTool.bat` 時，視窗會自動顯示您的 **手機存取網址**。只要手機與電腦連在同一個 Wi-Fi，輸入該網址即可存取。

### 2. 公網遠端存取 (Ngrok)
如果您需要在外面透過 4G/5G 網路存取：
1. 註冊 [Ngrok](https://ngrok.com/) 帳號。
2. 在 [Dashboard](https://dashboard.ngrok.com/get-started/your-authtoken) 取得您的 **Auth Token**。
3. 用記事本開啟 **`StartPublicAccess.bat`**，將 Token 貼入 `NGROK_TOKEN` 欄位。
4. 執行該批次檔，即可獲得一個全球通用的 **https** 網址。

### 3. 專屬抽選模式 (只看 B2)
在網址後方加上 `?onlyUse=B2`，系統會自動載入 B2 地圖並鎖定為唯讀的抽選模式。
